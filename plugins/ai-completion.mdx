## Codeium

[Codeium](https://codeium.com/) is a free AI-powered code completion tool that provides intelligent suggestions as you type. It's similar to GitHub Copilot but completely free.

### Installation

```lua lua/fastfist/packer.lua:150
use 'Exafunction/codeium.vim'
```

### Configuration

Codeium is configured with a simple keybinding in `after/plugin/lsp.lua:166`:

```lua after/plugin/lsp.lua
vim.api.nvim_set_keymap('i', '<M-CR>', 'codeium#Accept()', { expr=true, noremap = true, silent = true })
vim.g.copilot_no_tab_map = true
```

### Keybindings

<ParamField path="Alt+Enter" type="keymap" required>
  Accept the current Codeium suggestion
</ParamField>

<Note>
The keybinding uses `<M-CR>` which means **Alt+Enter** (Meta key + Enter) in insert mode.
</Note>

  <Step title="Trigger authentication">

## CodeCompanion

[CodeCompanion.nvim](https://github.com/olimorris/codecompanion.nvim) is an AI-powered pair programming assistant that integrates with various AI providers.

### Installation

```lua lua/fastfist/packer.lua:152-161
use({
    "olimorris/codecompanion.nvim",
    config = function()
        require("codecompanion").setup()
    end,
    requires = {
        "nvim-lua/plenary.nvim",
        "nvim-treesitter/nvim-treesitter",
    }
})
```

### Setting Up AI Provider

To use CodeCompanion, you need to configure an AI provider:

<Tabs>
  <Tab title="OpenAI">
    Add to your Neovim configuration:
    
    ```lua
    require("codecompanion").setup({
      adapters = {
        openai = {
          api_key = "your-api-key",
        },
      },
      default_adapter = "openai",
    })
    ```
    
    <Warning>
    Never commit API keys to version control! Use environment variables instead:
    
    ```lua
    api_key = os.getenv("OPENAI_API_KEY"),
    ```
    </Warning>
  </Tab>
  
  <Tab title="Anthropic (Claude)">
    Configure for Claude:
    
    ```lua
    require("codecompanion").setup({
      adapters = {
        anthropic = {
          api_key = os.getenv("ANTHROPIC_API_KEY"),
        },
      },
      default_adapter = "anthropic",
    })
    ```
  </Tab>
  
  <Tab title="Local Models">
    Use local AI models:
    
    ```lua
    require("codecompanion").setup({
      adapters = {
        ollama = {
          model = "codellama",
        },
      },
      default_adapter = "ollama",
    })
    ```
  </Tab>
</Tabs>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Codeium not showing suggestions">
    Check these items:
    
    1. Verify you're authenticated (restart Neovim if needed)
    2. Ensure you have an internet connection
    3. Check if the file type is supported
    4. Try manually triggering with `:Codeium Enable`
  </Accordion>
  
  <Accordion title="CodeCompanion not responding">
    Common issues:
    
    1. Verify API key is set correctly
    2. Check API provider status
    3. Ensure you have API credits/quota
    4. Check error messages in `:messages`
  </Accordion>
  
  <Accordion title="Alt+Enter not working">
    If the keybinding doesn't work:
    
    1. Check if your terminal supports Alt key
    2. Try using `<M-CR>` in terminal settings
    3. Consider remapping to a different key
    4. Test in different terminal emulators
  </Accordion>
</AccordionGroup>
